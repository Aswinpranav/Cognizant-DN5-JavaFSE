- Singleton Pattern
- Factory Method Pattern
- Builder Pattern

- Decorator Pattern
- Proxy Pattern
- Observer Pattern
- Strategy Pattern
- Command Pattern
- MVC Pattern
- Dependency Injection

Each exercise includes the Java source code and the corresponding output.
